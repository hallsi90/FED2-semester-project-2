# Semester Project 2

Auction web application built with Vite, TypeScript, and Tailwind using the Noroff Auction House API.

## Tech stack

- Vite
- TypeScript
- Tailwind CSS

## Getting started

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```
